Personal website and portfolio built on Next.js 16 App Router. It includes a homepage/hero, blog, projects, about, FAQ, tags, and learning pages. Content is sourced from MDX/contentlayer and data/*.ts files. Reusable UI lives in components/, routes in app/, longer layouts in layouts/, styling in css/ and route-local CSS files such as app/layout.css. The project started from tailwind-nextjs-starter-blog and has been customized into a portfolio/blog hybrid.
